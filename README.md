# Automated Robotic Arm

## Project Overview
This project is an ESP32-based automated robotic arm that picks and places objects using programmed instructions.

## Features
- ESP32 microcontroller
- Pick-and-place operation
- Automated movement
- Suitable for industrial automation and educational purposes

## Components Used
- ESP32
- Servo motors
- Robotic arm
- Power supply
- Connecting wires

## How It Works
The ESP32 controls the robotic arm by executing programmed instructions to move, pick up objects, and place them at the desired location.
